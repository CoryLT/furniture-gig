## Gotchas learned this session

- Native `<select>`/date pickers need `color-scheme` set on the theme to render dark — class
  styling alone isn't enough.
- Same-date rows have no inherent order from PostgREST; always add a `created_at` tiebreaker.
- A sale's transaction description is **`"Sold: " || title`** (so name search works on it).
- Stripe **live mode** has no "Send test event" button — verify via a real or 100%-off-coupon event.
- Server actions can be imported into client components (used for inline delete).
- Vercel reads env vars only at **deploy time** — adding one requires a redeploy.
