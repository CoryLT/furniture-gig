# Phase B Quick Reference

## API Endpoints

### Upload Photo

**Worker Profile:**
```
POST /api/upload-worker-gallery-photo
Content-Type: multipart/form-data

Body:
- file: File (image only, max 5MB)
- caption: string (optional)

Response:
{
  "success": true,
  "photo": {
    "id": "uuid",
    "file_path": "string",
    "publicUrl": "string",
    "caption": "string | null"
  }
}
```

**Flipper Profile:**
```
POST /api/upload-flipper-gallery-photo
Content-Type: multipart/form-data

Body:
- file: File (image only, max 5MB)
- caption: string (optional)

Response: Same as above
```

### Delete Photo

**Both Worker & Flipper:**
```
DELETE /api/delete-gallery-photo
Content-Type: application/json

Body:
{
  "photoId": "uuid",
  "type": "worker" | "flipper"
}

Response:
{
  "success": true
}

Errors:
- 401: Unauthorized
- 404: Photo not found
- 403: Not photo owner
```

## Components

### PhotoGallery
Reusable read-only or editable gallery display.

```tsx
import { PhotoGallery, type GalleryPhoto } from '@/components/ui/PhotoGallery'

<PhotoGallery
  photos={photos}
  isEditable={true}  // or false for read-only
  onDeletePhoto={handleDelete}
  userType="worker"  // or "flipper"
/>
```

**Props:**
- `photos`: GalleryPhoto[] — array of photos
- `isEditable?`: boolean — show delete buttons
- `onDeletePhoto?`: (photoId, type) => Promise<void> — delete handler
- `userType`: "worker" | "flipper" — for API calls

### PhotoUploadForm
Simple upload form with preview and validation.

```tsx
import { PhotoUploadForm } from '@/components/ui/PhotoUploadForm'

<PhotoUploadForm
  userType="worker"  // or "flipper"
  onPhotoUploaded={() => refreshGallery()}
/>
```

**Props:**
- `userType`: "worker" | "flipper" — determines which API endpoint
- `onPhotoUploaded?`: () => void — callback when upload succeeds

## Supabase Storage Path Structure

Photos are stored at:
```
photo-galleries/
├── {userId}/
│   ├── 1234567890-photo1.jpg
│   ├── 1234567891-photo2.png
│   └── 1234567892-photo3.webp
```

Public URL format:
```
https://your-supabase-url/storage/v1/object/public/photo-galleries/{userId}/{timestamp}-{filename}
```

## Common Tasks

### Load user's photos
```tsx
const { data: photos } = await supabase
  .from('worker_photo_galleries')  // or flipper_photo_galleries
  .select('*')
  .eq('worker_user_id', userId)    // or flipper_user_id
  .order('created_at', { ascending: false })

// Add public URLs
const withUrls = photos.map(p => ({
  ...p,
  publicUrl: supabase.storage
    .from('photo-galleries')
    .getPublicUrl(p.file_path).data.publicUrl
}))
```

### Upload via API
```tsx
const formData = new FormData()
formData.append('file', file)
formData.append('caption', 'My caption')

const response = await fetch('/api/upload-worker-gallery-photo', {
  method: 'POST',
  body: formData
})
const data = await response.json()
```

### Delete via API
```tsx
const response = await fetch('/api/delete-gallery-photo', {
  method: 'DELETE',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ 
    photoId: 'uuid',
    type: 'worker'  // or 'flipper'
  })
})
```
